Step Up Shoes is a full-stack online footwear ordering web application built using HTML, CSS, Node.js, Express.js, and MongoDB.
The website is designed to provide a smooth and responsive shopping experience where users can explore various shoe collections, view product details, and place orders online.

The frontend is developed with HTML and CSS to ensure a clean and user-friendly interface, while the backend uses Node.js and Express.js to handle server-side logic and API requests. MongoDB is used for storing product data, user information, and order details securely.

This project demonstrates the implementation of core e-commerce functionalities such as user authentication, product management, shopping cart, and order processing in a real-world web application.

Tech Stack: HTML, CSS, Node.js, Express.js, MongoDB
Project Type: Full-Stack E-Commerce Website
